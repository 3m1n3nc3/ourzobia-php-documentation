**************
Cpanel Webmail
**************

To configure the system to allow use of Cpanel Webmail, point your web browser to ``http://yourdomain.com/admin/configuration`` where ``yourdomain.com`` is your actual domain name or click on ``Management`` > ``Configuration`` > ``Main`` on the admin dashboard, and scroll down to the **Cpanel Configuration** section the rest should be self explanatory.
